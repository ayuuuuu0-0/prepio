# UI_ASSETS.md

# Purpose

This document defines the complete visual identity of Prepio.

Without this document AI agents will generate:

* Generic illustrations
* Generic dashboards
* Generic icons
* Generic onboarding

This document ensures every designer, developer, and AI agent builds the same world.

---
